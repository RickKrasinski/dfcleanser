# dfcleanser
Pandas Dataframe Cleanser is a Jupyter Notebook extension that lets you perform operations on a pandas dataframe.



## Chapters
Dataframe Cleanser has the following Chapters supporting specific functionality.
1) Data Import 
2) Data Inspection
3) Data Cleansing
4) Data Transform
5) Data Export
6) Data Utilities
