dfCleanser Census Utility


The Census utility allows you to retrieve census data and insert it into a pandas dataframe.

The census data included with this utility is derived from the US Census Bureau 2017 Data Release.


The Census data is broken into the following dataset groups for ease of use and can be downloaded and used independently of each dataset group.

To review detailed census info for each dataset and it's subsets click on the titles below:

  • Economic
  • Education
  • Employment
  • Health Insurance
  • Housing
  • Immigration
  • Internet
  • Population
  • Social
  • Transportation

Each dataset group is further divided into multiple datasets that contain the same columns but are indexd via the following index keys.

  • [City,State]
  • [County,State]
  • [State]
  • [Zip Code]

The indexing results into the generation of 4 datasets per dataset group and they are :

  • datasetgroup_cities
  • datasetgroup_counties
  • datasetgroup_states
  • datasetgroup_zipcodes

The process of getting and configuring census data is accomplished by clicking on the following:


  • Configure Census Data

Configure census data allows you to download and build the desired datasets.

The census data downloaded and configured is written locally to a dir off the dfcleanser package and will survive notebook close.

All census data downloaded is available across all notebooks.


Once the census data is downloaded and configured you can then do the following with the data.

  • Get Census Columns for df
  • Export the census data
  • Load Census Data to db(s)
  • Loading census data to a dfc df

Get Census Column(s) for df


The Get Census Column(s) for df functionality allows the user to build columns for any census column and insert the column into a user df.

The columns inserted are indexed based on zip code, city, county or state and the census index selected must match the corresponding index column in the user df.

Any user df index column value not found in the census df is set to Nan.

Parameters:
* census_index_type

The index type of the column names to insert.
Only the index types for datasets previously built are displayed.
When the index type is changed the index_column_list is cleared.

* df_to_insert_into

The df to insert census columns into.
The df_to_insert_into_cols field is updated when the df is changed.

* df_to_insert_into_cols

Columns from 'df_to_insert_into' to use as index values for the census data.

When a column is selected the selection is reflected in the 'index_column_list' field.

* index_column_list

The columns in 'df_to_insert_into' to use as census index columns.

Using the example above the following is used :

  • Chicago_df[City] is mapped to census dataset [city, ..] index field.
  • Chicago_df[State] is mapped to census dataset [.., state] index field.

You can also manually enter values in the 'index_column_list' to use as census index value.

Manually entered values must be encased in single quotes.

* nan_value

The nan value to use for columns without matching index.
The default of Nan is used if no value is specified.

Commands:  
Insert Columns into df(s)

Insert selected columns into a 'df_to_insert_into' indexed by 'index_column_list'.

Show Selected Columns
